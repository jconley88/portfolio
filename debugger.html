Debugging Tool for Ruby and Rails

I wrote my own ruby debugger.

I had been thinking about this project for years and finally decided to spend the first half of 2024 building out an alpha version. It is structurally similar to a time traveling debugger, but with a focus on higher level insight. It is composed of two separate code bases; a Ruby tracing gem and a VS Code extension.

It has reached a fully functional milestone and I am very excited to plan a public release after some additional stress testing. I'd be glad to share any aspect of this project and would love to hear from you.

Ruby Tracing Gem Technologies
  • Ruby, TracePoint gem, Parser gem, railtie and middleware, sqlite3
VS Code Extension Technologies
  • Typescript, sqlite3, tries
