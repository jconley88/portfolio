Mythos Character Sheet

A character sheet, inventory, and active battle tracker created solely for personal use.

Mythos is a mature but unreleased fantasy tabletop role-playing game similar to Dungeons and Dragons. I had the privilege of meeting the creator in person and later play-testing it online during the COVID years.

With a 100-page rule book and frequently changing conditions, recalculating stats by hand can be time-consuming and error-prone. I developed this tool to automate those calculations, allowing me to focus more on character decisions and gameplay. While UI perfection was not the primary goal, the tool excels in functionality and usability. I am eager to share more details about this project and welcome any inquiries or feedback.

  • Single-page Vue application with ~40 Vue components
  • Languages: JavaScript, HTML, CSS
  • Technologies: Vue, Vuex, Firebase, Web Storage API: localStorage, Zurb Foundation
  • Boilerplate styling with a focus on quick reference and usability
