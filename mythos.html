Mythos Character Sheet

A character sheet, inventory and active battle tracker created solely for personal use.

Mythos is a mature, but unreleased fantasy tabletop role-playing game similar to Dungeons and Dragons. I was lucky enough to meet the creator in person and later play-test online through the covid-years.

With a 100 page rule book and frequently changing conditions, recalculating stats by hand can take time and be error prone. I wrote this to handle that so I could better focus on character decisions and actual play. Touch base with me; I would love to share this with you.

  • Single page Vue application with ~40 Vue components
  • Languages: Javascript, HTML, CSS
  • Technologies: Vue, Vuex, Firebase, Web Storage API:localStorage, Zurb Foundation
  • Boilerplate styling with focus on quick reference and usability

I am proud of this, but it is not meant to be a UI masterpeice. I built this as a functional tool to learn Vue and have fun.
